## Structure du projet

Le code source se trouve dans le dossier `code`.

## Compilation

Le projet utilise **mavenCentral** comme dépôt Gradle afin de pouvoir travailler hors de l'IUT.

> **Sur les PC de l'IUT :** décommenter les lignes correspondantes dans `gradle.properties` avant de compiler.

La configuration du serveur se fait dans la classe `Config`.
